## 프로젝트 소개

방방곡곡 프로젝트는 한국관광공사가 운영하는 국내 여행 정보 서비스 '대한민국 구석구석' 사이트를 참고하여 만든 **대한민국 관광지 정보 제공 홈페이지**입니다.
공공 데이터 포털 Open API에서 국내 관광지 정보를 이용해서 지역별 여행지, 음식점, 축제 정보를 제공합니다.
